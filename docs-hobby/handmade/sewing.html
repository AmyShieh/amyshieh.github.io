Sewing

设计(审美&创造力) => 定款(表达&呈现) => 制版(结构&空间) => 纸样(专项技能) => 裁剪(手艺) => 缝制(手艺) => 装饰(审美) => 整理

备忘录​

技巧​

  • 打褶：张力 7，线迹调大。

基础​

解读纸样​

结构分辨: 按位置摆放 确定裁片份数: 前置罗列 确定物料: 细节(拉链等), 扣子, 面料（主料， 辅料）等 纸样标识:

  • 裁剪线：包含缝缝 1cm
  • 净样线：缝制时候的线
  • 布纹线：确定是横纹还是竖纹之类的。
  • 折线：折叠&熨烫(熨烫的时候尽量选择布料的背面)
  • 对称： T 恤 中间不剪开， 但衬衫中间要剪开，所以 T 恤会有对称标志，但是衬衫因为有扣子啥的，就不会有左右对称标志。
  • 剪口：一般用来对位
  • 省：腰围，胸围
  • 褶： 裙子，裤子之类的

缝纫机 & 锁边机​

针： 有凹槽的一边是左边 面线：一定是从左到右穿。 底线：旋转梭壳的螺丝来调节松紧度。（一般来说，拉着可以带起梭壳不会下落，甩一下落下，松紧就差不多。） 梭壳和梭芯：左手拿梭壳， 右手拿梭芯。 右手边，线朝下，放进梭壳后， 把线从梭壳的缺口里拉出来，再拉到小孔里。 拉东西线，呈现逆时针转动说明装对了。 准备工作：把面线和底线都放到后方去。

工业机：高速运转，线迹比较美观，功能单一，占据面积大 家用机：面积小，功能多，线迹没有工业机那么完美。不适合批量生产，锁扣眼。 调节针距，调节针的位置&配合压脚，z 字型线迹，锁扣眼，自动穿线，自动剪线，简易锁边基本就可以（额外买个定轨，制带器 ）。

面料​

天然纤维​

一般不会太亮 手感舒适

棉【 C 】：柔软透气，手感舒适，自然光泽，易皱，耐牢度不佳，易霉变。 毛 / 羊毛【 W 】：弹性好 马海毛【 M 】 兔毛【 R 】 真丝【 S 】： 亚麻【 L 】 羊绒【 WS 】 桑蚕丝【 MS 】

化学纤维​

人造棉【 R】： 涤纶【T】：易洗快干，不吸汗 锦纶【N】： 腈纶【A】： 氨纶【SP】： 天丝【Tel】：

加工方式​

机织 针织

特性​

幅宽：确保布纹线和布的长度保持平行。（和幅宽保持垂直） 缩水率 色牢度 耐牢度 克重 支数（密度）：纱线的密度， 21， 40-60(常用), 80, 100 D：300D 数值越大，越重 T：数字越大，越密，越小，越稀

面料使用​

FDC 面料图书馆(深圳 & APP)

棉： 可以用来做 【连衣裙，衬衫，短裤，吊带衫等】 棉麻： 可以用来做 【手工，包包等】 针织面料：可以用来做 【T 恤，睡衣，内裤，宝宝衣服等】 毛圈：可以用来做 【卫衣，运动裤等】 斜纹棉：可以用来做 【裤子，裙子，风衣 80%棉 + 20%氨纶等】 纱卡： 休闲裤，风衣，马甲 条绒：可以用来做 【背带裤，外套，裤子等，不适合贴身穿着】 灯芯绒：相比条绒会更细 毛绒：冬季衣服，要双层面料 牛仔：裤子（一般是全棉比较多，如果有弹性的话，基本是添加了氨纶） 丝绒：一般是全涤纶（礼服，连衣裙） 毛呢：冬季大衣，背心裙 粗花呢：外套，半裙 麻纱：连衣裙，衬衫，质感垂感比较好，会比较上档次。 网纱：做小朋友的裙子比较多（抽细褶） 蕾丝：女装，比网纱手感好 色丁：一般用来做里布 螺纹：衣服袖口，领口的地方 塔丝隆：羽绒服，冲锋衣，滑雪服，运动外套

线迹​

  1. 线条笔直： 空车练习，纸练习，穿线练习，弧线的时候，可以把针距调到最小的来练习。
  2. 面线和底线的松紧度调整： 弹框拧紧 => 面线调紧
  3. 针距适当：3 公分里面 10-12 针（面料厚），12-14 针（面料适中），14-16 针（面料比较薄）。调的数值越小 => 越密
  4. 平整度
  5. 倒针加固:按着倒车按键
  6. 固定间隔

流程​

服装用料​

裤子：裤长 * 2 + 缝份。一般幅宽 150 用的会比较多 喇叭裙： 裙长 * 4 + 缝份 百褶裙： 裙长 * 2 + 缝份 衣服：衣长 * 2 连衣裙：衣长 + 袖长 + 缝份

轮廓制版​

a. 尺寸 b. 轮廓 c. 结构 d. 元素 e. 领子 f. 袖子

排料​

手工排料 软件排料

  1. 长度、幅宽，经纬纱向
  2. 布料的正反面（在反面排料，正面比较光滑，有光泽，正面会有斜纹的肌理，布料边缘的针孔 凹陷的一面属于正面）。需要在正面做标记，免得到时候分不清正反
  3. 布料的纹路
  4. 布料的图案（有规则 & 无规则、图案朝向）
  5. 布料的缩水、耐热、色牢度
  6. 布料的毛倒向（顺毛，逆毛）

铺布方式：正面朝下，反面朝上 节约性：先主后次，先大后小，紧密套牌，缺口合并，合理拼接 对称性：对称的部分，折叠面料（可单层，对折，折边） 方向性：纸样布纹线与布料长度方向一致，最大偏斜量 3% 纹路性：根据纹路排料，纹路一致，可双向排料

前置工序​

  1. 烫衬、锁边
  2. 省、褶
  3. 分割线
  4. 口袋
  5. 拉链

缝合流程​

缝合工艺​

  1. 平缝
  2. 劈压缝/分缉缝
  3. 固压缝 / 坐缉缝
  4. 分压缝
  5. 搭接缝
  6. 来去缝
  7. 明包缝
  8. 暗包缝

主要的缝合类型​

  1. 短裙 （3 片，前片 + 后片 * 2） 后中缝 拉链 侧缝 腰头 下摆

  2. 裤子（4 片，前片 _ 2 + 后片 _ 2） 外侧缝 内侧缝 翻到反面，裆部缝合，前裆到后档 腰头 裤脚

  3. 上衣（4 片，前片 + 后片 + 袖子 * 2）【袖子和侧缝的缝合顺序可以根据实际的版型调整】 前后肩片肩缝 侧缝 袖子位置（袖子（侧缝） || 边缘处理） 袖子+衣服（绱袖） 领口处理 下摆处理

  4. 连衣裙（4 片） 前后肩片肩缝 衣身侧缝 袖子位置（袖子（侧缝） || 边缘处理） || 绱袖 领口处理 两边侧缝 下摆处理 上衣和下摆缝合（找到侧缝开始）

后置工序​

闭合件（锁扣眼、钉纽扣、盘扣、勾扣） 装饰配件（腰带、蝴蝶结、耳仔）
