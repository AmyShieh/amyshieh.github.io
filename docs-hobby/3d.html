软件基础操作​

  • 渲染模式： 材质捕捉

  • 显示设置：

    • 平滑阴影: 面数相对较少的时候依然保持一个平滑的外观

  • 导出格式：

    • glTF: 所有数据全部打包(颜色，材质等)
    • OBJ: 三维格式（无灯光，贴图等）
    • STL: 3d 打印格式
    • PLY: 网页端用的比较多
    • Nomad: 软件中所有的数据

  • 渲染(输出一张截图)

    • 建模
    • 打光（灯光）
    • 上材质
    • 后期调色

  • 添加基本体: 任何你想要的模型，都是由基本体或基本体的组合演变而来。

  • 模型操作

    • 体素合并： 连接处是软的
      • 相加（两个都可视）
      • 相交（两个都不可视）
      • 相减（保留的部分可视，减去的部分不可视）
    • 布尔运算： 连接处是锋利的

一些快捷键​

删除： X 鼠标中键： 拖拽模型 shift + 鼠标中键： 拖拽场景 shift + A：添加【网格、曲线、灯光等】 选中模型： 按下 G, 模型就会跟着鼠标走。 按住 X,则沿着 X 轴移动，按住 Y，则沿着 Y 轴移动，按住 Z，则沿着 Z 轴移动。 Alt + G： 回到初始新建的位置。 按下 S, 进入缩放状态。按住 XYZ 同上。 Alt + S： 撤销缩放（最初版） 按下 R, 进入旋转状态。按住 XYZ 同上。 Alt + R： 撤销旋转（最初版）

具体是按照世界坐标还是局部坐标，需要看你最后的 xyz 按了几下。 第一次是局部，第二次是世界坐标

H: 隐藏一个物体 Alt + H: 显示隐藏的物体

Shift + H : 隐藏没有选中的物体 Shift + D: 移动并复制

1： 正视图 3： 侧视图 7： 顶视图 9： 反转当前视图

2,4,6,8 每按一次 旋转 15° 5: 切换正交视图和透视视图 0: 切换到摄像机视图 .: 拉进视角

按住 ALT 键，鼠标中键转，或者，按~键，切换视图。

Ctrl + 空格： 最大化当前界面

Shift + S：出现一个 panel 目录。

按住 Ctrl 移动时，会有吸附的功能，一格动一下。

基础概念​

游标​

  • 当【游标】工具被被激活时，鼠标点击哪里，游标就会在哪里，新建的模型就会在哪里。
  • Shift + 鼠标右键： 移动游标
  • shift + C 回到世界原点

布局​

脚本=> 新建 => 常规 => 布局 可以对布局进行重置
