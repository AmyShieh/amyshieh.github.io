种花花 🌸​

不同品种的花都有其自己的生长(开放)周期。 可以在不同的季节种不同的花，一年四季都有花看！ 种花方面，我自己喜欢水培，因为感觉要比土培好看，但在养护方面其实水培要更复杂点~ 具体的可以根据自己的情况来选择。

春​

2 - 3 月

  • 🌷 郁金香
tip

种植到发芽: 15-25 天。 温度 5~15C 发芽。10-20C 生长开花光照 喜温睡通风，耐寒不耐热，怕干早，忌暴晒。 注意：11 月-次年 3 月播种期,夏季种球休眠期是不发芽的。[种球储存在阴凉干燥透气处]

4个片段
1. 剥皮，将种球表面深色外皮剥掉(注意别伤到白色内球)
2. 用半包生根粉或多菌灵如图兑水搅拌均匀后浸泡种球30分钟左右
3. 清洗浸泡过的种球，擦拭干净。玻璃瓶、石子也清洗一下。
4. 种球放在石子上，加自来水(最好晒1-3天)。根部在水里就可以，切记不可水位太多淹没种球。
5. 5天左右换一次水。如果出现发霉现象正常，把发霉擦掉即可。如发臭及时换水
1. 剥皮，将种球表面深色外皮剥掉(注意别伤到白色内球)
2. 用半包生根粉或多菌灵如图兑水搅拌均匀后浸泡种球30分钟左右
3. 将种球芽头/尖头朝上放到土上(选用松软透气的营养土/曹炭土均可)
4. 用土盖住种球, 深埋3-5厘米。长大后不容易歪倒。不过漏出点芽也行，不影响。
