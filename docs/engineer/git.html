basic​

commit vs commit -a​

-a:

  1. git add
  2. 自动删除在 index file 中，但是不在工作树中的文件
  3. commit

结论： git commit -a === git add . && git commit

merge​

git merge命令的语法是，将指定分支合并到当前所在分支。 所以把 a 分支的内容合并到 b分支时，需要切到b分支，执行git merge a。
