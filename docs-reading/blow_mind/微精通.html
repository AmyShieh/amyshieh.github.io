微精通

作者：罗伯特·特威格尔

虽然这个世界似乎要迫使我们相信，一生只应该做一件事，但我们其实可以扔掉这个观点，从小做起。

学习的本质是观察， 它能让我们清楚如何抓住每一步骤的重点所在。

微精通旨在让学习变得更容易，而非困难，因为学习中最大的障碍就是耗费时间太长，造成半途而废。

人类是学习型动物，不再忙着学习时，就会忙着死区—— Bob Dylan

学习中遇到的最大困难，是如何把握每一部分的相对重要性。观察别人时，我们能根据他们的表现和举止来弄清楚每项元素到底有多重要。

关于一项技能的掌握上，花费大量的时间经常被提及，通常是 1w 小时，这很容易让人起疑心，因为光靠时间是不够的，学习和精进技能的态度才是至关重要的。

捷径总是魅力十足，但绝非坦途。通常需要更多的精力，最终把自己搞的精疲力尽。

思考​

我的思考

最近在读这本书的时候，刚好在学习水彩画画，看到一个评论说的是，你最开始要做的是，是把结构画正确，细节都是最后的东西了。 如果结构画错了，那后期，你再怎么努力，作品都是不尽如人意的，反而白瞎了你自己的时间。 早上在上班路上，刚好想起一个笑话，可能不是特别相关，但冥冥之中又觉得有点关联：一条狗，走在一条未干的水泥路上，没走一步就踩一个坑。评论是，有时候，方向错了，你多走的每一步，都会是你日后挨揍的理由。 做东西，学技能，同样也是应该遵循 MVP 原则，你得知道骨架是什么，然后确定自己方向的正确性，再去往下钻研。

不要去小看一点点的改变，只要你是走在正确的道路上，你现在的行为，比你之前的好，你整体的发展趋势就是向好发展。

Action​

关于微精通​

无法实现目标最大的原因是半途而废、失去动力、心烦意乱。 你可能以为自己很顽强、很独立，但实际上，我们从开始学习时，就需要得到回报来激励自己，尤其在尝试新事物，并进行了一段时间的时候。 如果在学习过程中不能获得一系列微小的成功，你将失去信心，甚至放弃，特别是在完全自学的情况下。

内核​

每一种微精通都有精确的结构：

  1. 入门技巧
    • 所有的微精通在实践初期都会存在障碍，而入门技巧能够克服这些障碍。 有一些是需要增强自信心和熟悉度，有一些是注意力分配程度。入门技巧能让你快速地对事物有基本把握，虽然到最后阶段，你可能不需要它了，但在初期，它是你最好的朋友。
  2. 协同障碍
    • 量化技能的核心元素，给每个元素打指标分，分开去练习。
  3. 背景支持
    • 良好的装备，时间，和开放思维。克服一切的外界阻力。
  4. 成功回报
    • 明确的，可视化的成果，而非是一个含糊的。
    • 斯蒂夫·查普曼是教练和创造力讲师，他向公众宣布自己的挑战，用可能蒙羞的危险来驱使自己勇往直前。这是用恐惧来客服懒惰和注意力不集中的独创性案例，用一种消极来战胜其他消极。(讲真的，我其实一直很佩服这样的人，有足够的勇气，这让我想起高中时代有个很漂亮的女生，成绩也不错，高三的时候，她发了一条朋友圈，大概是说，我要是文科第一的身份站在演讲台怎么怎么样，最后她也没有成为文科第一，但是这种勇气，我是觉得，很厉害，我有很多次想要去尝试，但是发现我自己做不到，说到底，也许还是太在意别人落在自己身上的目光？我不知道)
  5. 可复验性
    • 作者每次去咖啡店都会画出里面的杯子，勺子，时间久了，就会提升自己的观察力。
    • 可游戏性 ，不要让自己做的东西 那么枯燥。
  6. 可实验性
    • 突破已有的规则，就好比 J 形划艇的秘密其实更接近于 C 型，由此你可以发展为 L 型，Z 型等，做不同的尝试，直到直到适合你自己的。

动态学习​

阅读书本是一种静态学习，而一些实践性强的技能，是没有办法靠着书本学会的。但是，如果你找到了其中的门道，是可以通过书本来提升自己的技术水平。 静态学习遵循一定的步骤顺序，而动态学习关注步骤间的关联，以及每一步骤该分配多少关注度。

锻炼记忆​

因为电子设备，互联网的发展，我们的长、短期记忆都因之受损。但是大脑用进废退。越是不同，就会认为你不需要这项技能了。

遗愿清单​

愿望清单听上去会比遗愿清单更积极。但这样的清单 ，才是改变生活的工具。 我们可以认为生活是由外部驱动的。也可以认为自己有更多的主动权：在我们衰弱到无力做任何事之前，简单列出自己希望完成的事情。

心流​

心流处于焦虑与过度控制之间。一开始就要 百分百专注，因为分心是致命的（虽然心流 不受时间影响，但是时间压力会阻止它的产生）。最后你要反观自己的内心，焦虑，无聊？还是活力满满？

技能​

画画​

重点在于观察，而不是动手去画

演讲​

  1. 马上真实描述你的所思，所感，所想，所做。演讲不是简单的闲聊，与你的精神状态和演讲主题都密切相关。
  2. 演讲中途卡壳了，要马上请教听众。
  3. 尽量放松，和听众积极互动，不去绞尽脑汁回忆演讲稿内容。
  4. 一有机会就主动演讲，酒吧里，聚会上都行。大家看会觉得有点奇怪，但这是增强信心和提升幽默感的方法。而自信和幽默通常是成功演讲的关键。
  5. 公开演讲可能会带你进入即兴表演这个让人惊叹的领域。在即兴表演中，要努力让合作伙伴表现优秀，而不是炫耀自己。大家都尽力成就对方，这世界将多么神奇而美好。

书中的一些观点​

  • 学习微精通时要知道，所有的东西都围绕着中心点运转，找到中心点是自信心的源泉，也能让你更容易理解微精通。显然，这比其他因素都重要的多。
  • 只有那些具有陡峭学习曲线的东西才值得学习。进入这样的学习时，你必须静下心来，不要再给自己找借口，就像学游泳，要么沉下去，要么浮上来，而人类是擅长游泳的。
  • 任何人都可能建议读教科书，但是你能否以微精通的视角来读，这很重要。
  • 有时人们也会对画画感兴趣，但那都不是真正的微精通，因为他们涉猎太广。
  • 一个简单的事实是：观察别人学的最快。
  • “看上去有难度”到底是什么意思？我觉得是一种为了让自己心安理得的宽泛视角，意思是，你不准备尝试解决或弄清楚任何问题。你等着好点子自己蹦出来，而不是逼着自己就所研究的事物挖掘想法。
  • 微精通要求我们接纳有限的、独立的活动，这有助于深度观察。不是像在一个广阔的领域里四处游荡，我们可以拉进镜头，集中注意力观察。比起视图把形形色色的东西一股脑儿塞进心不甘情不愿的大脑，这样学习要快得多。
  • 微精通的每一次实验，都是又一次练习而已。我们要相信，它并非是“一次性买卖”。
  • 以牺牲其他自我为代价，来过度发展某种自我毫无益处。这种自我可能是为了服务于某种组织，你可能是一只优秀的“工蜂”，但这不是你来到这个星球上的理由。商业自我想要压制其他的任何自我。
  • 如果过度开发脑力，到头来你可能会轻视他人，这是同理心发展失败的结果。倘若对他人关心过度，你最终可能会阻碍他人习得自立能力，这是洞察力的失败。很明显，过度发展某方面是荒诞的。

一些参考数据​

  • “在任何进攻或防御时，都要找到自己的物理重心，这样就提高了平衡性，进而增强了自信心，然后更确信自己的重心到底在哪里，这是个有利于进步的良性循环。相对的就是恶性循环： 你的自信动摇了，找不到重心，从而失去了平衡，更加削弱了自信心，导致表现不佳。”——合气道八段大师 雅克·佩耶
  • 神经系统功能的最新研究表明，大脑的绝大部分由多感知神经细胞组成。也就是说，一类脑细胞能处理多种感觉，如嗅觉，视觉，而不是一一对应。所以，我们学习时，用的维度、感官越多，效果越好。
  • 大脑塑造是一个物理过程。灰质实际上能变厚或缩小，相应地，神经连接可得到锻炼并增强，也可能被削弱甚至切断。
  • 从生物进化的角度看，生活导向的微精通获得起来更“自然”。大脑遵循用进废退的原则。
  • 同一时间被激活的神经元间的练习会被强化。如果你第一次去巴黎旅游时遇到下雨，这两件事就会总被联系在一起。
  • 经验越丰富，越多样化，对感觉的影响越大，事情在大脑中留下的印象越深刻，大脑连通性也就越强。这不但能增强记忆，还能防止衰老。
  • 如果总是重复做一件事，大脑中已有的神经网络会得到强化，但是不会产生新的神经通路。也就是说，中、短期记忆的使用越来越少。
  • 直觉经验最近才得到科学的认可：记忆不被使用就会衰退。如果不去需要自己确认各种方位的新位置，比如记商店位置，找返回酒店的路，甚至记停车位，你会逐渐失去认路的基本技能。把记忆作为一种技能来训练，对人是有帮助的。
  • 有一种传统而过时的观点：每个人都有自己独特的“学习方式”。这暗示着，如果不用自己喜欢的方式学习，你是学不到所需知识的。这也让你有接口不努力学习新东西。在《软连接 Soft-wired》中之初，大脑的可塑性可延伸到我们的学习方式上，换句话说，我们可以学会如何学习。
  • 显摆能力，也是一种社会需求，承认它。每个人都需要关注，只是有人多，有人少而已，每个人也都在学着吸引别人的注意力，只是有人做得好，有人做得差而已。
