真希望你也喜欢自己

房琪

在家中我也需要每天努力，才能赢得太太和孩子的尊敬与爱。——李安 没有什么是理所应当，即便是血缘中的纽带，相濡以沫的情分，仍然要去培养呵护。就这种谦卑和诚惶诚恐，源于教养，善良和智慧。——鲁豫对李安的采访。

人生中，比起互相搀扶着成长，单枪匹马的闯荡才是常态。 而旅行，做不做攻略，都能到达目的地，但想要选择什么样的过程，还得自己去探寻和下决定。

生活毕竟不是电视剧，一个念头的出现，也不可能成为逆袭的开始。

站在高处望深渊，坠入深渊识攀爬。人不活一个点，人活起伏。——长安十二时辰。

瘦，是美的唯一标准吗？肚子上有肉就代表着自我放弃，双下巴出来就是不自律，这些究竟是来自谁的定义？真正的自由，不是叛逆，也不是放纵和无所顾忌，而是学会由内而外地接受自己，当然也包括接受自己的不完美。

周围比较优秀的女性，都有一个共同的他特点：很喜欢自己，很欣赏自己。这体现在，喜欢自己的价值观，欣赏自己的人格魅力，喜欢自己的为人处世的态度和方法，欣赏自己做的决定。

对于主角来说，爱与崇拜，唾手可得，而作为一个小人物，一个生活里，很平凡的人来说，每一份尊重都要靠自己赚得。

人的本性，都是喜欢听表扬，不喜欢听批评，但是成长之所以会让人变得不一样，不正是我们我们要把这些本性变得更符合生活发展的方向，更加契合成长的轨道吗？

"若批评不自由，则赞美无意义"。

迟迟钟鼓初长夜，耿耿星河欲曙天——白居易

我也知道，自己并没有偶像剧女主角的光环，我平凡得连跑龙套的路人甲都算不上，想要被人看见，就必须站在有光的地方。

勇敢地承认，你在乎！野心不是也不该是一件难以启齿或羞于表达的事。

承认自己平凡，也承认自己不甘平凡。

"人之患，在好为人师！" 我们要做的不是说服别人，而是尊重彼此。

"我比赛不是为了打败其他选手，不是为了滑得比别人好，而是要做到自己百分之百最好" ——谷爱凌

"只有足够努力，你才配得上你的野心。"在它燃烧时，我们要小心，别人这份热烈烫伤他人，只把它放在心里，为自己提供能量就好。

所以，别为拥有野心而感到抱歉，总有一天，我们不用再追着光奔跑，因为，我要这光，就为我而来。

"只有你足够与众不同，你才能足够被需要"——马东

面对恶意：两岸猿声啼不住，轻舟已过万重山。

"人生当中成功只是一时的，失败才是主旋律，但是如何面对失败，却把人分成了不同的样子。"——2014 巴西世界杯比赛，西班牙队 VS 澳大利亚队。

这个时代，风口常有，却不是每个人都能成为飞上天的猪。

当遇到挫折时，如果不去想着，怎么爬起来，而是一味从别人身上找原因抱怨这个世界，那么此刻，就是坠入深渊的开始。

"无论你喜欢与否，生活都是一场比赛，懦夫从未启程，弱者死于途中，强者就必须前行，一刻也不能停！"

想赢这个世界？只需要你站起来的次数比倒下的次数多一次就行。

我们这一生中，需要用金钱来捍卫尊严的时刻实在太多了。

比起拥有什么样的父母、爱人、家庭、背景，你拥有的能力和内心，才是支撑你走到最后的内核。

在我们还没有足够的能力策马扬鞭时，千万不要去责怪为何风没有吹向你。

并不是所有人都能够足够幸运，可以在二十多岁的年纪找到一生笃定的奋斗方向。我们不过是平凡的大多数，

"被讨厌的勇气并不是要去吸引讨厌的负向能量，而是，如果这是我生命想绽放出最美的光彩，那么，即使有被讨厌的可能，我都要用自己的双手双脚往那里走去。"——被讨厌的勇气。

  1. 千万不要不好意思谈钱。
  2. 赚钱的目的是，让你有些钱可以不用去赚钱。没有绝对的自由，面对生老病死，总会有捉襟见肘的时刻，但钱确实可以让你，不去做那些你不想做的事情。
  3. 你是真的赚不到自己能力之外的钱的。
  4. 不要相信一夜暴富的神话。别着急，真的不要太着急。

"大家总是在讨论站着赚钱还是跪着赚钱，但其实都不是，成年人往往是商量着挣钱"——李诞

不知道做什么？迷茫？

  1. 多花点时间和精力去进行自我分析和自我思考
  2. 在真正走入社会之气，多去试错，在不知道自己要什么之前，可以先知道自己不要什么
  3. 不轻易评判别人的喜好和选择
  4. 永远不要提别人的人生做决定。

无论选择哪一条路，都是在为 i 自己的人生画一张越来越清晰的画。

我不被消费主义绑架，但是我也需要有取悦自己的能力。
